### 강의 정리 - 305. 그룹 만들기

<br />

같은 레벨의 도형들은 그룹을 지을 수 있다. 사각형 두개를 그룹지어보겠다. g 태그를 사용하여 안에 같이 감싼다.

```html
<style>
  .shapes {
    width: 600px;
    height: 400px;
    // svg 안의 도형은 fill로, 도화지는 background로
    background: #ddd;
  }
  .rect {
    fill: orange;
    stroke: dodgerblue;
    //px, em 다 가능
    stroke-width: 10;
  }

  .ellipse {
    /* fill: #fff000; */
  }

  .group-1 {
    // 요소 각각에 효과를 준 게 더 우선순위가 높다.
    fill: yellow;
  }
</style>

<body>
  <svg class="shapes">
    <g class="group-1">
      <rect x="10" y="20" width="200" height="100"></rect>
      <rect x="50" y="170" rx="10" ry="10" width="100" height="100"></rect>
    </g>
    <circle cx="350" cy="250" r="30"></circle>
    <ellipse
      cx="200"
      cy="400"
      rx="100"
      ry="50"
      fill="red"
      stroke="green"
      stroke-width="20"
    ></ellipse>
  </svg>
</body>
```
